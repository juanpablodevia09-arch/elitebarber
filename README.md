# EliteBarber

Plataforma moderna para la gestión de barberías.

## Características

- Inicio de sesión
- Registro de clientes
- Panel para barberos
- Gestión de citas
- Interfaz moderna
- Integración con Supabase

## Tecnologías

- React
- TypeScript
- Tailwind CSS
- Vite
- Supabase

## Instalación

```bash
npm install
npm run dev
